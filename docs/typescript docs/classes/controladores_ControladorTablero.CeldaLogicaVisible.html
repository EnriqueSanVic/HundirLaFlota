Class CeldaLogicaVisible

Clase para representar una casilla lógica del tablero visible de un jugador.

Una celda lógica visible tiene en su estado un símbolo (char) que es el valor que guarda y que representa el símbolo que aparecerá en el tablero gráfico.

Un booleano para ver si ha sido tocado por que había un barco y no era una posición vacía ante un ataque. Un booleano para ver si ya ha sido descubierta por el enemigo o no.
