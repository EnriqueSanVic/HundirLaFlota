Class TableroGrafico

Componente gráfico.

Este es el componetne de un tablero, por medio de polimorfismo se pueden generar tableros de tipo real (tablero que puede ver el jugador dueño del tablero) o de tipo visible (tablero que pueden ver los contrincantes de un jugador).
